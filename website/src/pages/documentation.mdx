# Welcome to Carrot Documentation

*C*onvenient *a*nd *r*eusable *r*apid *O*MOP *t*ransformer: bringing together health data from across the UK.
Carrot consists of two tools: [Carrot Mapper web tool](mapper) for generating mapping rules, and
[Carrot Transform ETL tool](transform) for applying mapping rules to data.

At a glance, the diagram below shows how Carrot Mapper and Transform function in the process of mapping data to OMOP CDM.

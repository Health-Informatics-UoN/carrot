# Requirements

## System

| Requirement       | Minimum Specification                                        |
| ----------------- | ------------------------------------------------------------ |
| CPU               | 4 vCPUs                                                      |
| Memory (RAM)      | 8 GB                                                         |
| Container Runtime | OCI-compliant container runtime (Docker, Podman, Kubernetes) |
| Database Access   | PostgreSQL 14                                                |

## Vocabularies

Carrot Mapper requires OMOP vocabularies, which can be downloaded from [Athena](https://athena.ohdsi.org/), to be installed in the database. Please follow the instructions in [OMOP CDM configuration section](/mapper/deployment/local#omop-cdm-configuration)
or [OMOP schema](/mapper/dev_guide/quickstart#omop-schema-and-vocabs-setting) to set up these.
